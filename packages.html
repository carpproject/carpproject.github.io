Pre-built packages for PENCIL tools and VOBLA compiler

  • PENCIL tools deb
  • PENCIL tools rpm
  • VOBLA compiler deb
  • VOBLA compiler rpm
Requirements:
  • Scala compiler (scalac) at least 2.10 version in system PATH.
  • Antlr3 library (antlr3.jar) in system CLASSPATH.
  • VOBLA compiler requires PENCIL tools package.
