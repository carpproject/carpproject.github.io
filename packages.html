Pre-built packages for PENCIL tools and VOBLA compiler

  • PENCIL tools deb
  • PENCIL tools rpm
  • VOBLA compiler deb
  • VOBLA compiler rpm
Requirements:
  • Scala compiler (scalac) at least 2.10 version in system PATH.
  • Antlr3 library (antlr3.jar and ant-antlr3.jar) in system CLASSPATH.
  • VOBLA compiler requires PENCIL tools package.

Ebuilds

CARP project maintains overlay for the PENCIL tools and VOBLA compiler. Layman config for the overlay can be copied below: 
<?xml version="1.0" ?>
<repositories>
    <repo name="CARP" quality="experimental" status="unofficial"
          type="git" src="git://github.com:carpproject/carp-overlay.git">
    <link>git://github.com:carpproject/carp-overlay.git</link>
<description>Kayrick Gentoo Overlay</description>
</repo>
</repositories>
